CSC 120 Lab 1

This lab will introduce the NetBeans IDE (Integrated Development Environment) for Java programming, and the programming process of declaring, instantiating, and using objects from pre-existing Java classes.

Part 1: Use File Explorer to create a storage folder for your Java projects for this course.

You need a common place on your computer or the Mount Union network, or on the computer you will be working on regularly, to store all of your work for this class.  We suggest that you use your H: drive on the Mount Union network (VPN access is required off campus), because our IT department backs up all student work space regularly, so in the event of a device failure you will be able to recover your work.  But you can choose to store your Java programs anywhere that is convenient for you.

The following steps will guide you to create a folder on your H: drive on the campus network step-by-step.  As long as you create a CSC120 folder somewhere for your files, you can have this folder anywhere that fits your working style.  Find a different location that you prefer and use it in place of the H:\ drive in steps 2 and 3. For consistency throughout the instructions in this course, we will always make reference the H: drive.

  1. Open File Explorer.

  2. Browse to your H: drive, which is your personal file space on the campus network.

  3. In the folder H:\, create a new folder and name it CSC120 (choose File | New | Folder from the menu, then type CSC120 (uppercase C, uppercase S, uppercase C, the digit 1, the digit 2, the digit 0, with NO SPACES!) and press the [Enter] key.  Then double-click on the CSC120 folder name in the right pane.

  4. In the CSC120 folder you just made, create a new folder and name it Java (choose File | New | Folder from the menu, then type Java (uppercase J, lowercase all other letters, with NO SPACES!) and press the [Enter] key.

  5. Double-check that you have a CSC120 folder that contains a Java folder in the storage location that you used (if you followed these steps, the location will be your H: drive).  All future instructions will refer to your CSC120\Java folder when asking you to copy files or otherwise work with your Java projects.

  6. Minimize Windows Explorer by clicking on the underscore symbol in the upper right hand corner of the window; do NOT click on the X - that would close the application instead of "setting it aside".
